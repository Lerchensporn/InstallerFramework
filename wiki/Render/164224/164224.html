Versionen

Hier werden die Änderungen in den verschiedenen Versionen angegeben.

Version 1.0

Diese Version ist das erste Release des InstallerFrameworks.

Funktionen in dieser Version
  • der Installer funktioniert mit dem .Net Framework 3.5
  • abstrakte Schnittstelle für Installer
  • Installer für Registrierungseinträge
  • Installer für Ereignisprotokolle
  • Installer für Dateien und Verzeichnisse
  • Installer für Registrierung der Software auf dem System
  • Installer für Dienste
  • Installation, Deinstallation und Rollback werden unterstützt
  • mehrstufige Benutzeroberfläche
  • Ermitteln der Fortschrittsstufe der Installation
  • Finden von installierter Software auf dem System
  • schnelle Installation
  • vollständiger Verzicht auf „Windows Installer“
  • gute Erweiterbarkeit mit eigenen Komponenten
  • sehr gute Dokumentation der Frameworks

Bugs und Zielstellung für die nächsten Versionen
  • [BUG] das GUI wird während der Installation blockiert
  • Updates installieren
  • Abbrechen, Anhalten und Fortsetzen der Installation
  • Unterstützung von Multithreading (für bessere Nutzung der Systemleistung, schnellere Installation, keine Blockierung der GUI)
  • Ausführung von Aufgaben beim Neustart (falls z.B zu ändernde Dateien von System genutzt werden)
  • Sicherung der Installation in einer temporären Datei (falls das System abstürzt)
  • Installer für Dateiregistrierung
  • Komprimieren der zu installierenden Software
  • Registrierung von Dateien
